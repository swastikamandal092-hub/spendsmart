# 💰 SpentSmart – Expense Tracker App

SpentSmart is a full-stack expense tracking web application built using **Node.js, Express, MongoDB, and EJS**.  
It helps users track income and expenses, manage personal finances, and visualize spending patterns.

---

## 🚀 Features

- 🔐 User authentication (Signup/Login/Logout)
- 👤 Personalized dashboard for each user
- 💸 Add, edit, and delete expenses
- 📊 View income and expense analysis (graphs)
- 🧾 User profile management (update name, age)
- 🔒 Protected routes (only logged-in users can access data)
- 🧠 Ownership validation (users can only access their own data)

---

## 🛠️ Tech Stack

- **Frontend:** EJS, HTML, CSS, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** Passport.js (Local Strategy)
- **Session Management:** express-session
- **Validation:** Joi
- **Others:** method-override, ejs-mate
